# AutoOS

The ultimate local-first desktop automation platform. AutoOS combines advanced visual intelligence with a zero-latency direct execution model to automate your computer in plain English.

## 🚀 Key Features

- **Visual Intelligence**: Take screenshots and get detailed hardware/system summaries.
- **Direct OS Execution**: Regex-based local command parsing for instant, lag-free performance.
- **Antigravity Planner**: High-confidence intent routing for non-technical users.
- **Comprehensive Hardware Control**: Manage Wi-Fi, Bluetooth, Storage, and Power modes with automatic settings navigation.
- **Browser Automation**: Full web control via `browser-use`.
- **Privacy First**: Everything runs locally on your machine.
